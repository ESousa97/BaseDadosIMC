•Quando a loja reportar que o cupom não está sendo impresso, mesmo após a configuração da impressora térmica, devemos validar a configuração do SAT.


1. Primeiro passo é verificar se o SAT está respondendo. Abra o Ativação Sat ou Ativação Sweda que fica no menu inciar.




2. No sistema de ativação sat ir na opção buscar Sat e clicar status SAT.



3. Para consultar o status mais detalhado digitar o código de ativação e clicar no status operacional.

Segue o código:

Frango Assado - Viena - Batata Inglesa - KFC - PH e OG: 09060964

Postos de combustiveis: cliente@1234

Se nenhum dos códigos acima deu certo, tente: 00000000



4. Se o SAT estiver Ok ele trará um relatório quando selecionado Status Sat ou Status Operacional.

Pelo Status operacional e possível validar se o equipamento está bloqueado ou não.



Caso SAT esteja com Status Bloqueado necessário acionar Jose, Ruan, Gustavo, ou no grupo da IMC pelo celular do SD, para que os mesmo validem o equipamento.


5. No gerenciamento de dispositivos é possível validar os drives e a configuração do SAT.



6. Validar os adaptadores de rede se o SAT está disponível, caso ele não apareça será necessário efetuar a atualização do drive.



7. Caso ele não esteja configurado será necessário identificar o dispositivo para atualizar o drive.



Ou se estiver dessa forma, ele está sem o drive instalado.



8. Neste caso é necessário transferir o drive de instalação de acordo com sistema operacional (32 ou 64).

Para atualizar clique com o botão direito no dispositivo selecionado e escolha a opção updatedrive.



9. Coloque a opção que você deseja procurar o drive no computador.



10. Selecione a pasta para onde o drive foi transferido.



11. Efetuar a instalação.



Instalação efetuada com sucesso fara com que o SAT apareça na listagem do gerenciador de dispositivo.



15. Realizar novamente o teste no Ativação Sat ou Ativação Sweda.
