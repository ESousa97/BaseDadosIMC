ALIVE POSTO – EXTRAIR ARQ. XML


INSTRUÇÕES:


Localizando os Arquivos:


Os arquivos por padrão ficam armazenados em “C:\AliveIT\XML SAT”.


Clique com o botão direito em cima do atalho do SAT/Posto e selecione “Abrir local do arquivo” e volte uma pasta.




Para cada mês existe uma “pasta/diretório” informando o “ano-mês”.

Exemplo: a pasta 202106 são os arquivos do mês 06 do ano 2021.

Dentro de cada pasta/diretório do “ano-mês” existem várias pasta informando o dia.



Dentro de cada pasta/diretório do dia estão os arquivos XMLs do SAT.



Compactando os Arquivos (pasta/diretório ano-mês):

Para enviar os arquivos para o escritório/contabilidade é recomendável compactar a pasta antes de enviar.

Obs.: use um software compactador de arquivos de sua preferência, no exemplo estamos usando o WinRAR https://www.win-rar.com/download.html

Selecione a pasta, clique com o botão direito e clique em “Add to ‘nomedapasta’.rar”.



Será criado um novo arquivo com extensão .RAR, envie esse arquivo utilizando um método de sua preferência (whatsapp/e-mail).
