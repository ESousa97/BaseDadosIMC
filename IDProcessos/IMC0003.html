TELEFONIA - VOIP

As lojas utilizam aparelhos voip, exceto o KFC

Se a loja reclamar que não está recebendo ligação, ou se a linha esta muda precisamos validar se o equipamento está aparecendo no Advanced IP scanner e pedir para loja a foto do visor do aparelho para validar o aparelho está com conexão.

Existem casos em que a loja colocou o aparelho em não perturbe , se estiver dessa forma no visor vai aparece a sigla DND , nesse caso a loja só precisa aperta a tecla sem identificação.

Existem dois tipos de aparelho Yealink ou siemens, e normalmente o final do IP da telefonia é 100.

(existem casos em que ainda está fora do padrão)

OBS: se aparecer um aparelho cisco a loja está utilizando o aparelho incorreto (nesses casos devemos acionar a IMC para trocar o aparelho).



Para validar se o mesmo está com rede , abrir o navegador no servidor da loja e digitar o ip do aparelho na url de sites

LOGIN: ADMIN ou admin

SENHA: ADMIN ou admin



Se ao digitar o ip não carregar a tela acima , provavelmente o problema está no aparelho, necessário validar os cabos , se mesmo assim o aparelho não voltar pedir um field no local para verificar

OBS: validar se antes do IP esta HTTP

Na aba Account precisa verificar o status registro :

Registred - a linha está ok

Register faled - alinha está com problemas , nesse caso pode ter 3 possíveis problemas :

Senha incorreta , o servidor com o ip errado ou o transporte errado





Register faled – transporte errado

O padrão é que o campo transporte esteja com UDP

Se tiver UDP e mesmo assim não estiver funcionando validar a informação do servidor está correta, o ip do servidor deve ser 10.11.4.80






Se estiver com o IP certo e estiver com UDP e mesmo assim não funcionar , trocar o UDP para TCP

Se mesmo trocando para TCP não funcionar, digitar novamente a senha no campo password

SENHA : Union!@1234

OBS: validar se o DSN está igual ao DNS da máquina, na aba network

Se com todas essas validações o problema persistir, solicitar a visita de um field para verificar a estrutura da onde o aparelho está: se o ponto está funcionando , se a tomada não está em curto e etc

Caso o field não encontre problemas devemos acionar o Rodrigo.
