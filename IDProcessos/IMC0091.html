WINDOWS UPDATE NÃO ATUALIZA - ERRO 80072EE2



● Erro apresentado nesse processo ocorre quando o WindowsUpdate não consegue se conectar com os servidores da Microsoft ou ao WSUS(Windows Server Update Services.).





● Após identificar o erro, teremos duas formas de resolvê-los, primeiro identifique se o terminal que você está acessando tem o ServicePack 1 instalado.

● OBS.: O processo para quem não tem o ServicePack 1 instalado vai está mas abaixo com o nome em vermelho..





● Após verificar, acesse o seguinte link Catálogo Microsoft Update.





● Assim que o site abrir, procure a seguinte atualização KB3138612, caso seu Windows seja 64bits, baxei a opção marcada em amarelo, caso seja 32bits, baixe a com nome Atualização do Windows 7 (KB3138612).





● Assim que vocês clicarem em Transferir, irá abrir uma janela, onde devem clicar no link dentro dela para realizar o download.





● Ao baixar, esperem concluir para executar o arquivo.





● Assim que executarem, irá aparecer a seguinte tela de instalação, nessa parte somente aguardar a conclusão.





PROCESSO PARA QUEM NÃO TEM O SERVICEPACK 1 INSTALADO NO EQUIPAMENTO.



● Ao acessarem o Catálogo do Microsoft Update, vão procurar a seguinte atualização KB976932 caso o Windows seja versão 64bits, baixar a marcada em vermelho, caso contrário, baixar a primeira opção.





● Ao clicarem em Transferir irá aparecer uma outra tela, onde devem clicar no link presente.





● Ao baixar somente executar, após a instalação em algumas máquinas é pedido parar reiniciar.



● Após reiniciar o equipamento, voltem no windows Update, irá aparecer uma opção escrito "Verificar Atualizaçãoes Online" onde mostra a imagem abaixo.





● Caso não apareça, somente clicar em Procurar Atualizações, e aguardar.



● No caso apresentado, deu certo.





● Nessa parte somente clicar em instalar atualizações, e aguardar a finalização.
