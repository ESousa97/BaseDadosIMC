DEGUSTUPDATE NÃO ENCONTRADO



COMO RESOLVER?

1. Acessar o servidor da loja

2. Acessa a pasta DegustWin (que geralmente fica na raiz do Windows Disco Local C, mas não é uma regra)

3. Achar o arquivo DegustUpdate (.exe)



4. Copiar este arquivo

5. Colocar na pasta DegustWin do terminal (que geralmente fica na raiz do Windows Disco Local C, mas não é uma regra)

6. Acessar o sistema Degust novamente no terminal

Se após execução destes passos o erro persistir, seguir o seguinte fluxo

1. Realizar abertura de chamado na LINX

2. Comunicar o Jose/Ruan/Gustavo do Problema com a informação do chamado aberto na LINX
