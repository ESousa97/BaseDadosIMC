Touch não funciona - terminal modelo P1530



● O processo de touch não funciona descrito abaixo só server para modelos NCR P1530, peça foto do terminal para loja, e se for igual do modelo abaixo podem seguir.

● Caso seja com o sistema Aloha, abra o CMD como administrador, e digite o seguinte comando Systeminfo e aguarde.







● Se o modelo do terminal com problema se enquandra conforme imagens acima, podem seguir normalmente com o processo.



● Baixem o seguinte driver no equipamento: 3M_Capacitive_7.13.8.

● OBS.: Sempre coloquem os arquivos baixados na pasta TI_IMC dentro do disco C do equipamento para manter um ambiente organizado.





● Assim que baixarem, pode extrair o arquivo, entrem na pasta que foi extraída e executem o seguinte software _3MCap.





● Ao executar, irá apresentar a seguinte tela, onde podem clicar em Next.





● Na tela seguinte, aceite os termmos e clique em Next.





● Vamos clicar em Next novamente.





● Selecionem a opção Typical e vamos avançar.





● Nessa parte vamos clicar na opção Touch digitizer uses Windows' built-in touch functionality.





● Next novamente.





● Pronto somente aguardar a finalização da instalação, após finalizar, reinicie e teste o terminal.





● Caso seja preciso realizar a calibração do touch, faça o processo TOUCHSCREEN - CALIBRATION.
