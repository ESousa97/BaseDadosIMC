UPLOAD DE VENDAS DO DIA


Após a finalização do fim do dia, deverá realizar os seguintes passos


1. Clicar no botão de Comunicação


2. Visualizar o item 3. “Gerar arquivo de movimento”



3. Selecionar a data e verificar

a. Se o arquivo estiver com a informação Exportada = SIM, significa que o movimento foi realizado b. Se o arquivo estiver com a informação Exportada = NÃO, significa que o movimento não foi realizad

4. Caso o cenário seja o item 3.b, de arquivo pendente, selecionar o item número 2 “Enviar arquivos de movimento pendente”



5. Aguardar o envio dos arquivos

6. Validar novamente verificando no item 3. “Gerar arquivo de movimento”

Obs: Se a loja teve muito movimento no dia, as vendas tem até as 20hrs para subir.
