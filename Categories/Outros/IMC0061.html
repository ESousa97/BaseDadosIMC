INSTALAÇÃO SITEF WEB


Primeiramente, acesse a máquina que requer a instalação ou reparo do software "Sitef Web".


Após obter acesso, verifique a versão do Java no sistema Windows da máquina. Abra o prompt de comando (CMD) e digite "java –version" ou "java --version", dependendo da versão do Windows; um dos comandos funcionará corretamente.


Caso a versão do Java seja 271, é possível pular a etapa de instalação e seguir diretamente para o passo 7, para realizar a configuração dos links dentro do Java.


Bem vamos ao primeiro passo, verificar qual a versão do Java instalado no equipamento.
