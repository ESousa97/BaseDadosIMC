Validando Equipamento SAT



● Procedimento realizado abaixo serve para todas as lojas que possuem o equipamento SAT.



● Primeiramente, acesse o terminal que tenha equipamento, a versão do software para teste é a 3.0.3 ele possui o seguinte nome Ativação SAT.

● Caso não tenha, realize a instalação. Ativacao_SAT_Sweda_3.0.3





● Sempre abra como administrador.





● Ao abrir, vamos até a opção Consultas SAT.





● No campo Cod. Ativação vamos digitar o código 09060964 caso seja Aloha ou Degust, se for posto de combustiveis o código é cliente1234.





● Ao digitar, vamos clicar em Status SAT.





● Essa mensagem aparece caso o sat esteja comunicando normalmente com o terminal, agora vamos clicar em Status Operacional.

● Rosa: Mensagem de resposta se o SAT estiver OK ou não.

● Laranja: Mostra o numero de serie do equipamento SAT.

● Vermelho: Informa o IP do SAT.

● Azul: O gateway do equipamento.

● Amaralo: Mostra o DNS do SAT.

● Verde: Se ele está conectado a rede normalmente.





● Se descerem mas um pouco na mesma tela, verifiquem também se ele apresenta essa informação de Status Operacional Desbloqueado





● As luzes do sat devem ser avaliadas, nele temos.

● Power: Significa que o SAT está sendo alimentado.

● Rede: Se ela estiver ligada, significa que o sat está conectado corretamente.

● Sefaz: Valida a comunicação do sat com a SEFAZ.

● AC: Valida a comunicação do SAT com a USB do terminal.

● CFe: Se estiver ligada, significa que tem cupons que não foram transmitidos.





● Abaixo segue explicações caso aconteça no sat que estão atuando, para conseguirem identificar onde está o problema.





● Explicações sobre a luz de REDE.





● Led CF-e.





● Nessa situação peçam um video de 10seg para loja.





● Realizado todas as verificações, caso o sat esteja operante sem problemas, vamos voltar para o software Ativacao SAT SWEDA.

● Vamos na opção Configurar Rede, dentro dela vamos clicar em Carregar.





● Devem obter a resposta igualmente na imagem abaixo.

● OBS.: Caso apresente uma mensagem, relatando que a configuração foi excluida, devemos configurar um novo IP igual o passo após esse.





● Vamos voltar na opção Consultas SAT, clicamos novamente em Status Operacional, e verificamos, caso apareça a seguinte informação NAO_CONECTADO como ilustra a imagem abaixo, vamos configurar um novo IP no equipamento.





● Vamos voltar no sevidor da loja, abrir o Advanced IP Scanner e clicar em Exibição.





● Dentro da opção Exibição vamos marcar a opção Mostrar desconhecido.





● Vai abrir diversos IP's desconhecidos, vamos pegar um que esteja enquadrado próximo ao equipamento SWEDA INFORMATICA LTDA que no caso esse seria outro sat em REDE, para mantermos o padrão da loja, vamos pegar um IP mas próximo dele.





Configuração do IP no Ativação SAT

● Após obter o IP, acesse a aba "Ativação SAT" e então vá para "Configurar Rede".


● Insira o IP obtido na caixa "IP SAT".


● A Máscara padrão deve ser configurada como 255.255.255.0, independente do IP da loja. Esta máscara sempre será utilizada.


● O Gateway deve corresponder ao range de IP da sua loja, terminando em .1. Nunca configure o Gateway com o IP de qualquer equipamento, para evitar a queda imediata da rede da loja.


● Dns é padrão para todos os equipamentos, com exceção do computador da gerência ou administrativo.


Seguindo esses passos, a configuração será realizada com sucesso.





● Após preencher corretamente, podem clicar em Enviar, irá apresentar a seguinte mensagem abaixo.





● Vamos novamente voltar para a aba Consultas SAT e clicar em Status Operacional.





● Verifique se todas as informações no SAT estão OK conforme a imagem acima.

● Scrolla a pagina, e verifique também as informações abaixo.





● Agora vamos na aba Ferramentas e clicamos em Atualizar.





● Caso dê tudo certo, irá apresentar a informação conforme a imagem abaixo, caso dê SAT em processamento, realize o seguinte procedimento antes de continuar Sat em processamento, procedimento deve ser seguido até a parte de reinstalação do driver.





● Caso apresente Erro desconhecido tente realizar a extração das logs do sat.





● Ao aparecer abaixo, desça a barra de rolagem até embaixo, pois as informações mas recentes estarão lá.





● Você pode realizar a análise das logs apartir desse momento.





● Caso prefiram, podem clicar dentro da log, apertar Ctrl + A do seu teclado e depois Ctrl + C e copiar as logs para seu bloco de notas, para melhor análise.





● Prcure informações sobre SEFAZ, para saber se o sat tem comunicação ou algum erro, verifique se o SAT sincroniza o NTP.







● Todas essas informações coletadas ajudaram a saber se o equipamento SAT está ok ou não.
